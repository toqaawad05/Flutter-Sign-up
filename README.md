  <h1>📄 Sign Up Page – Flutter UI</h1>

  <p>This Flutter project implements a responsive <strong>Sign Up page</strong> using clean UI design and Material widgets. It demonstrates the use of layout widgets, input fields, custom styling, and structured code design in Flutter.</p>

  <hr>

  <h2>📱 Features</h2>
  <ul>
    <li>Clean and modern user interface</li>
    <li>Centered layout using <code>Column</code> and <code>Center</code></li>
    <li><code>AppBar</code> with centered title and custom font style</li>
    <li>Multiple <code>TextField</code> inputs with rounded borders</li>
    <li>Custom <code>ElevatedButton</code> with styling</li>
    <li>Use of <code>Padding</code>, <code>SizedBox</code>, and <code>OutlineInputBorder</code> for proper spacing and design</li>
    <li>Ready to be extended to connect with Firebase or backend for authentication</li>
  </ul>
